Assignment 2 Readme:

In order to compile the program, traverse the command prompt or similar terminal to the directory where this file is located.
Once completed,compile the program by typing in the following to the command prompt:

							javac Anagram.java
							
Once the program has successfully compiled,  program to run the program type this into the command prompt (while still staying in the same directory):

							java Anagram inputfile.txt outputfile.txt
											
Where inputfile is the .txt file you would like to sort and check for anagrams and,
			
outputfile is the name of the .txt file you would like the sorted Anagram linked list to be written into
